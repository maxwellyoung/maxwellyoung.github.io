What are the differences between relative, absolute, and fixed positioning?

18.04.22

Positioning is a key part of CSS web design. It allows you to change the behaviour of different elements on the page whether text, an image, card etc. There are in total, 5 position types. These are "static", "absolute", "relative", "fixed" & "sticky". "Static" is the default or normal position type. Absolute
